# 怪物设计

## 核心哲学

```
怪物 = 移动的压力源，不是要你清完的目标。

你需要保护盒芯不被怪干碎。
你可以选择杀怪，也可以选择不理它快速推完箱子。
两个选择都对——取决于你手上有什么牌。
```

---

## 基础数值

```
共享HP（玩家 & 盒芯共有）：
  ┌──────┬──────┬──────┐
  │      │ 攻击 │ 血量 │
  ├──────┼──────┼──────┤
  │ 玩家 │  4   │  80  │
  │ 盒芯 │  0   │  80  │
  └──────┴──────┴──────┘

  盒芯（带球关卡的球箱）：ATK 0 / HP 80
  玩家：                ATK 4 / HP 80

  HP 是共享的——盒芯受伤 = 玩家受伤 = 同一个 80 池子。
  玩家攻击力 4 用于：
    - 基础移动撞敌（不耗卡，4 伤）
    - 冲撞牌的基础伤害锚点（具体牌的攻击力待定，应 ≥ 4）
  盒芯攻击力 0（不会攻击）。
```

## 四大阵营：桌面游戏的统一主题

玩家拿国际象棋牌，敌人来自另外三家桌面游戏：

| 阵营 | 桌面游戏 | 核心机制 | 关键词 |
|------|----------|----------|--------|
| 国际象棋 | Chess | 冲撞七式 | 玩家侧（已设计） |
| 围棋 | Go | 连子强化 + 对撞 | 断、连、围 |
| 麻将 | Mahjong | 联动 combo | 碰、吃、杠 |
| 扑克 | Poker | 牌型搭配 | 对、顺、同花 |

三个阵营分布在 Run 的不同阶段，每段换一种战术逻辑——不是换皮怪，是换玩法。

## 基础行为模型

```
每回合（玩家行动后）：
  每个怪物向盒芯移动 1 格（最短路径，A* 或 BFS）
  贴到盒芯 → 攻击 → 伤害计算 ↓

伤害流程：
  盒芯有格挡值 → 先扣格挡，格挡吸收等量伤害
  格挡归零后 → 剩余伤害扣共享HP
  共享HP归零 → 盒芯碎裂 → 本关失败

怪物不能推箱子（箱子对怪物是障碍物）。
怪物不能穿过普通墙（幽灵类除外）。
```

### 移动优先级

```
1. 向盒芯移动（最短路径）
2. 如果路径被墙/箱子挡住 → 绕路
3. 如果完全无法到达 → 向玩家移动（攻击玩家也是伤害盒芯，共享HP）
4. 贴脸后原地攻击，不移开
```

## 围棋兵（提案一）

### 取材

```
黑白两色的围棋兵小人。
圆润的石子身形，正面有单色（黑/白）。
在网格上形成"棋串"。
```

### 连子规则

```
同色围棋兵在正交方向（↑↓←→）相邻 → 连通为"棋串"

棋串中每个棋子的面板 = N / N
  N = 该棋子所在棋串的总棋子数

  1 子孤立 → 1/1
  2 子相连 → 2/2
  3 子相连 → 3/3
  N 子相连 → N/N
```

### 对撞规则

```
玩家冲撞敌人 → 玩家 ATK(4) ≥ 敌人HP → 敌人死，无事发生
                  玩家 ATK(4) < 敌人HP → 无事发生（撞不动）

没有反伤。常态下所有对撞都是单向的。
撞不死 → 敌人还在，下回合它继续打盒芯。这就是天然的惩罚。

围棋兵的威胁模型：
  1 只 3/3 放着不杀 → 每回合打盒芯 3 伤
  杀了它 → 花了一回合行动，不掉血，少挨 3 伤/回合
  不杀它 → 推盒芯叠盾硬抗，但盾在衰减

抉择是回合经济问题：
  "这回合是去清怪，还是去推盒芯叠盾？"
```

### 切断手段

```
1. 冲撞穿杀：撞死棋串中的棋子 → 串自然断开 → 两侧重算
2. 推箱阻断：把箱子推到两个围棋兵之间 → 正交邻接被挡住 → 串断开
3. 骑士践踏：跳过障碍踩死棋串中的连接节点
4. 火炮翻越：翻过前排直接杀后排的连接节点
```

### 战术分析

```
长处：
  - 规则极度简单（连N就N/N，一句话讲完）
  - 自动制造战术深度（杀第一个吃N伤 vs 先断后杀）
  - 与推箱子网格天然统一（围棋也是网格，箱子=落子断连）
  - 视觉可读性极高（看到一串同色就知道威胁等级）

短处（待验证）：
  - 连子太长时N/N数值可能爆炸（比如10连=10/10，一拳打残玩家）
    → 缓解：地图上同色围棋兵的总数可控，小房间放5~6个
  - 需要AI考虑"维持连接"的行为，增加寻路复杂度
    → 缓解：怪物不需要主动维持连接，玩家断连是被动的
```

## 与冲撞系统的交互总表

```
怪物在冲撞路径上 → 触发对撞

  战车冲撞：直线穿透，路径上每个怪物依次对撞
  骑士践踏：跳过路径，落点有怪 → 对撞，不伤路径上的
  主教冲撞：同战车，斜线方向
  火炮冲撞：翻越第一单位（不伤），后续怪物正常对撞
  士兵冲撞：斜角一格，该格有怪 → 对撞
  主帅践踏：周围 3×3 所有怪物 → 同时对撞（各算各的）
  皇后冲撞：8向任选，同战车规则
```

## 与格挡系统的交互

```
怪物攻击盒芯 → 先扣格挡，格挡不够再扣共享HP

  推箱子 = 叠格挡 = 创造安全窗口
  安全窗口内 → 可以安心对撞杀怪（HP不掉）
  格挡衰减中 → 开始有压力
  格挡清空后 → 必须快速处理威胁

不同威胁等级的怪物，制造不同的"格挡消耗率"：

  1/1 小怪 → 每回合消耗 1 格挡，不紧急
  4/4 连子围棋兵 → 每回合消耗 4 格挡，必须优先处理
  高伤精英兵 → 格挡蒸发器，逼你立刻重新推箱
```

## 生成规则

### 数量

```
N = floor(曼哈顿距离 / 4) + 难度修正

  难度修正：
    普通关：  -1
    标准关：   0
    高难关：  +1
    BOSS关：  +3
```

### 放置

```
- 怪物在地图边缘生成，不在房间内部初始出现
- 初始位置距离盒芯至少 5 格
- 同色围棋兵初始可以相邻放置（形成初始棋串）
- 不同阵营不混在同一关（围棋关全围棋、麻将关全麻将）
```

### 刷新

```
是否支持关卡中途刷新怪物？
  - 可从地图边缘定时刷出新怪（持续压力）
  - 也可以全量初始放置（固定威胁）
  → 待定，取决于想要的压力曲线
```

## 难度递进

```
前期（第1~3关）：
  → 围棋兵，少量（2~4只），单色
  → 玩家学习连子机制和断连对策

中期（第4~7关）：
  → 围棋兵数量增加，出现双色
  → 或引入麻将兵（如果麻将设计完成）

后期（第8~10关）：
  → 扑克兵精英
  → 多颜色/多花色混合，复杂威胁

BOSS关：
  → 特殊BOSS单位（单只，高血量？阶段变化？）
  → 待设计
```
