# 卡牌设计

## 核心动词：冲撞

战士只有一个动词：**冲撞**。分两层。

### 基础移动（无卡）

```
↑↓←→ 走一格。就是"一格的冲撞"。

  敌人 → 撞死
  箱子 → 推一格
  空地 → 走过去
  墙   → 不破。走不动。

贴身敌人免费处理，不耗卡。
但墙后的敌人——你够不着，必须交牌。
```

### 冲撞卡（耗牌）

```
战车/骑士/主教/火炮/士兵/主帅/皇后。

规则同冲撞，但多了：穿透、多格、破墙。
每张卡打开一种"线"。
```

### 核心体验：瞄线

```
每回合你做一件事：看棋盘，画一条线，决定冲不冲。

线画对了 → 一冲多杀，推球到位
线画错了 → 堵住，下回合被反击

不在线上 → 冲不到 → 只能基础走位重新瞄线

墙是核心障碍：
  贴身敌人 = 不需要卡
  墙后敌人 = 必须交卡
```

## 冲撞体系

### 战车冲撞（直线）

```
方向：上下左右四选一

冲撞模式（战士碰到）：
  空地 → 穿过，继续冲
  敌人 → 撞死，继续冲
  箱子 → 进入推箱模式，推着走
  墙   → 撞碎第一面墙，停

推箱模式（箱子碰到）：
  空地 → 推着继续走
  敌人 → 停。箱子不杀人。
  墙   → 停。箱子不破墙。

顺序处理，一格一格。

举例：
  战士 -> 敌人1 -> 箱子 -> 敌人2
    → 敌人1死 → 推箱 → 箱碰敌人2停
    → 结果：敌人1死，敌人2活，箱子停在敌人2面前
```

### 骑士践踏（跳跃）

```
方向：走"日"字

无视路径上的一切（墙、箱、敌人）。
落点格子：
  空 → 落脚
  敌人 → 踩死，落脚
  墙 → 可以穿过，但落点是墙 = 不能落脚（无法执行）
  箱子 → 落脚在箱子旁边（不可穿箱落地）

独立价值：唯一的越墙手段。
墙后的敌人，战车冲不到，骑士跳过墙去踩。
```

### 主教冲撞（斜线）

```
方向：四斜角选一

规则同战车冲撞，方向不同。
斜线上没有墙壁保护时，主教是最长距离的清线手段。
```

### 火炮冲撞（翻越）

```
方向：上下左右四选一

同战车冲撞，但遇第一个单位（敌或箱）时翻越过去。
翻越 = 该单位不受伤害。
翻越后继续以冲撞模式处理后续格子。

核心场景：
  玩家 -> 箱子 -> 敌人
    战车 → 推箱 → 箱碰敌停，什么都没死
    火炮 → 翻箱 → 撞死敌人

  玩家 -> 敌人A -> 箱子 -> 敌人B
    战车 → 敌人A死 → 推箱 → 箱碰B停
    火炮 → 翻敌人A → 跳箱子 → 撞死敌人B

火炮不穿透。翻越后撞到的第一个敌人会死，但不能再穿过。
它打的是"后排"。
```

### 主帅践踏（解场）

```
无位移。对自己周围 3×3（8 格）全部敌人造成伤害。

价值：冲进去被围后解场。
冲撞 = 纵向突破
践踏 = 横向解围
两个动词，两种处境。
```

### 士兵冲撞（短兵）

```
方向：四斜角选一
位移：仅一格
那格上有敌人 → 撞死
那格是空地 → 落脚
那格是墙/箱 → 不可执行

主教冲撞 = 斜线扫荡（宏观）
士兵冲撞 = 斜角一步（微观）

长线靠战车/主教。贴身斜角是它们的死角。
```

### 皇后冲撞（终极）

```
方向：8 向任选（战车四向 + 主教四向）

规则同战车冲撞。
终极瞄线牌，任何方向都能冲。
```

## 牌系总表

| 牌 | 方向 | 特殊规则 | 定位 |
|----|------|----------|------|
| 战车冲撞 | 上下左右 | 穿透杀敌、推箱、破墙 | 基石 |
| 骑士践踏 | 日字 | 无视路径 | 越墙 |
| 主教冲撞 | 四斜角 | 同战车 | 斜线长程 |
| 火炮冲撞 | 上下左右 | 翻越第一单位 | 打后排 |
| 士兵冲撞 | 四斜角 | 仅一格 | 贴身短兵 |
| 主帅践踏 | 无 | 身边 3×3 | 解场 |
| 皇后冲撞 | 8 向 | 同战车 | 终极 |

## 格挡：移即防

不需要格挡卡。防御来自移动本身。

### 规则

```
箱子被推动 X 格 → 获得 X 格挡值（累加到已有值上）

箱子停在原地：
  第 1 回合 → 格挡值不变
  第 2 回合 → 格挡值减半（取整）
  第 3 回合 → 格挡值清空
```

### 节奏

```
推球 = 攒盾
停一回合 = 盾满，可以打怪（白送窗口）
停两回合 = 盾减半，开始有风险
停三回合 = 盾清空，球裸奔

节奏：
  推 → 推 → 停(满盾打怪) → 推 → 推 → 停(满盾打怪) → ...
```

### 与冲撞的联动

```
推的距离 = 盾的厚度 = 安全预算

战车冲撞推很远 → 大量盾 → 但冲完后位置可能不好 → 你愿意用这个位置换盾吗？
火炮短冲 → 盾少 → 但位置好 → 下回合还能瞄好线

"冲多远"成了一个防御决策。
不是越远越好，是够用就好。
```

## 没解决的问题

- 各牌的伤害数值
- 稀有度与获取途径
- 初始牌组构成
- 牌组上限与删牌
- 每回合抽牌数、手牌上限
