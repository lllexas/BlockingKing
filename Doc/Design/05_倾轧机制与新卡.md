# 倾轧机制与新卡

## 倾轧（Overrun）

### 定义

倾轧是一种可叠加的层数型 buff。

> 拥有 **X 层倾轧**时，在一次冲撞的推箱阶段中，身前推动的箱子最多可以**碾碎 X 个敌人**。
> 每碾碎一个敌人，**倾轧层数 -1**。
> 倾轧层数跨回合保留，无自动清空机制。

### 与冲撞系统的交互

```
正常冲撞流程（无倾轧或倾轧层数为 0）：
  玩家 → 冲撞路径 → 进入推箱模式 → 箱子碰到敌人 → 停下

有倾轧时的冲撞流程：
  玩家 → 冲撞路径 → 进入推箱模式 → 箱子碰到敌人 →
    若倾轧层数 > 0 且冲撞伤害 ≥ 敌人 HP：
      → 敌人被碾碎（死亡）
      → 倾轧层数 -1
      → 箱子继续前进
    若倾轧层数 = 0 或冲撞伤害 < 敌人 HP：
      → 箱子停下（与原有规则一致）
```

### 伤害计算

碾碎伤害 = 触发本次冲撞的**来源伤害**

- 基础移动推箱碾人：伤害 = 玩家基础 ATK（4）
- 战车冲撞推箱碾人：伤害 = 战车冲撞的伤害（具体数值由卡牌决定）
- 其他冲撞牌同理

### 关键规则

1. **仅对推箱阶段生效**：倾轧只在"玩家推着箱子移动"时触发，玩家自身冲撞路径上的敌人由正常冲撞规则处理。
2. **逐格判定**：箱子每前进一格遇到敌人时独立判定，伤害不足则立即停止。
3. **不触发反伤**：碾碎是单向伤害，敌人不对玩家或盒芯造成反击。
4. **可断开围棋棋串**：碾碎棋串中的棋子后，该棋子死亡，棋串自然断开，两侧重算面板。
5. **层数显示**：UI 上需显示当前倾轧层数（buff 图标 + 数字）。

## 新卡：获得一层倾轧

### 基础信息

| 字段 | 内容 |
|------|------|
| **卡牌名称** | 铁蹄印记 / 获得一层倾轧（暂定） |
| **费用** | 1 |
| **类型** | 技能卡 / 增益 |
| **稀有度** | 普通 |
| **效果** | 获得 1 层倾轧。 |

### 设计意图

这是倾轧机制的基础获取卡，设计目标是：

- **低费用、低收益、可积累**：1 费换 1 层，不赚但稳定，鼓励玩家在前期慢慢攒。
- **教学卡**：让玩家第一次接触"倾轧"概念，理解"层数 = 可碾人数"。
- **构建基石**：在牌组中可作为倾轧流的核心过牌/积累组件。

### 使用场景示例

```
场景：玩家面前有 敌人A(1/1) → 箱子 → 敌人B(1/1) → 敌人C(2/2)

无倾轧：
  战车冲撞 → 撞死敌人A → 推箱 → 箱子碰到敌人B → 停下
  结果：敌人A死，B和C存活，箱子停在B面前

有 2 层倾轧：
  战车冲撞 → 撞死敌人A → 推箱 → 碾碎敌人B(-1层) → 继续前进 →
  碾碎敌人C时：伤害(4) ≥ HP(2)，但倾轧只剩1层？
  等等，敌人C是2/2，伤害4 ≥ 2，倾轧剩1层，可以碾！
  结果：敌人A/B/C全死，箱子继续前进直到撞墙停

实际：
  有 2 层倾轧：
    战车冲撞 → 撞死敌人A → 推箱 → 碾碎敌人B(-1层，剩1层) →
    继续前进 → 碾碎敌人C(-1层，剩0层) → 箱子继续前进到空地/墙停
    结果：A/B/C全死
```

## 倾轧机制相关设计备忘

### 后续可扩展的卡牌方向

| 卡牌概念 | 效果方向 |
|----------|----------|
| 铁甲重骑 | 本回合内，推箱自动获得倾轧（回合结束层数清空） |
| 碾骨护符（遗物） | 每击败一个敌人，获得 1 层倾轧 |
| 连环践踏 | 消耗所有倾轧层数，每有 1 层对前方直线造成 X 伤害 |
| 战车冲锋+ | 战车冲撞，若本次推箱碾碎了敌人，则返还 1 点费用 |

### 与现有系统的化学反应

| 系统 | 交互效果 |
|------|----------|
| **战车冲撞** | 推得远 = 碾得多，倾轧利用率最高 |
| **火炮冲撞** | 翻越第一单位后推箱碾后排，双重穿透 |
| **围棋兵连子** | 碾碎 = 自动断开棋串，不用特意冲连接点 |
| **格挡系统** | 倾轧鼓励推箱前进，推得远 = 盾厚，一举两得 |
| **步数限制** | 一牌清多怪 = 节省步数，在 StepLimit 模式中价值更高 |
