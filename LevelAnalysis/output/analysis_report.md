## 5. 拼贴策略建议

### 尺寸约束
- 关卡宽度范围 5-49，中位数 13 -> 拼贴后单行建议不超过 3-4 个关卡并排
- 宽高比集中在 0.8-1.6 (约70%) -> 横向/纵向混排兼容性好
- 面积中位数 143，均值 208 -> 分布右偏，注意大关卡节奏控制

### 难度曲线
- 63% 关卡为简单/中等 (≤8箱)，37% 困难 -> 可用简中难=4:3:3 做渐进难度曲线
- 相邻关卡难度跳跃建议 ≤2 档 (如 极简→很简→中等→偏难)
- 高潮段可从 Sasquatch VI/VIII/XI 选取 (30-50箱)
- 缓坡段从 Microban 簇选取 (1-7箱)

### 空间节奏
- 开阔关卡 (可通行>0.65) 占比约 28% -> 可穿插在大面积密集段后提供"喘息"
- 极密关卡 (墙密度>0.55) 占比约 9% -> 适合短关卡增加压迫感
- 墙体密度与箱密度正相关 (r≈+0.15) -> 紧密空间自然伴随更多箱子，节奏可统一控制

### 来源搭配
- Microban 1-3 高度相似 (欧氏距离<1.0) -> 可互换
- Microban4 与 Original 接近 -> 可做 Microban→Original 过渡
- Sasquatch VI/VIII/XI 自成一簇 -> 保留为硬核池，不与其他混合
