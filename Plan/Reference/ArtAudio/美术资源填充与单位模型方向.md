# 明日 美术资源填充与单位模型方向

优先级：P1

原因：当前系统已经能跑，明天的关键是让评审看到一个统一、可读、连续的产品画面。单位模型是战斗识别的核心，优先级高于 BGM。

## 当前视觉语言

现有画面更接近：

- 霓虹战术沙盘
- 电子棋盘
- 数字化桌游
- 规则件/棋子/投影单位

不建议转向写实仓库、木箱、自然地砖。单位模型应继续服务“棋盘规则件”的视觉语言：轮廓清楚、俯视可读、颜色语义明确、动画不复杂。

## 已统计单位

### 玩家侧

- 玩家 `player`
- 普通箱子 `box`
- 核心箱子 `Box.Core`

### 目标/地形

- 普通目标 `target`
- 核心目标 `Target.Core`
- 敌人出生点 `Target.Enemy`
- 墙体 `Wall`
- 不稳定墙 `Wall.Unstable`

### 敌人

代码中的敌人视觉分支：

- `Enemy.Go`
- `Enemy.Grenadier`
- `Enemy.Crossbow`
- `Enemy.Artillery`
- `Enemy.CurseCaster`
- `Enemy.Guokui`
- `Enemy.Ertong`

当前 `DrawSystem` 会按 `EntityBP.name` / tag 区分这些敌人。

## 模型方向头脑风暴

### 玩家

目标：可控单位，第一眼必须认出。

已定方向：

- 国际象棋棋子方向，优先用“兵”。
- 轮廓要简单、稳定、俯视可读。
- 色彩建议白色/青白色，和敌人区分。

备选 primitive 方案：

- 圆柱底座
- 短柱身体
- 球形头
- 可加一点青白发光材质或底座光圈

避免：

- 过细人形
- 和敌人一样的黄/橙色块

### 普通箱子

目标：推箱子核心对象，必须厚重、可推、可堆。

方向：

- 发光数据方块
- 半透明能量货箱
- 棋盘上的方形模块
- 简单 cube + 边框 + 面板纹理

### 核心箱子

目标：escort 生命核心，必须比普通箱子重要。

方向：

- 普通箱子上加核心发光球/晶体
- 更高的玻璃罩
- 青白/金色核心灯
- 有“护送目标”感的能量核心

### Go / 围棋子

目标：基础敌人，最普通、数量最多。

已定方向：

- 黑色或深色围棋子
- 如果没有合适模型，直接做扁球/短圆柱。
- 顶部可加红/橙色眼点或小发光点。
- 简洁低矮，移动感强

语义：基础步兵。

### Guokui / 锅盔（一筒麻将牌）

目标：厚一点、移动慢一点或特殊敌人。

已定方向：

- 一筒麻将牌方向。
- 主体用立方体/长方体。
- 面上嵌一个圆柱体或圆形发光片，表示“一筒”。
- 可竖立，也可略微倾斜，保证俯视能看到圆点。
- 比 Go 更大、更像精英棋子。

语义：装甲/中型敌人。

### Ertong / 二筒

目标：可分裂/特殊行为敌人。

已定方向：

- 二筒麻将牌方向。
- 主体同 Guokui，用立方体/长方体。
- 面上嵌两个圆柱体或圆形发光片，表示“二筒”。
- 两个核心灯要清晰，暗示分裂成两个 Guokui。
- 颜色比 Guokui 更粉/紫，和普通敌人拉开

语义：分裂体/不稳定单位。

### Grenadier / 掷弹兵

目标：远程爆破/投掷单位。

已定方向：

- 中国象棋炮系棋子，但使用石字旁的 `砲`。
- `砲` 更像投石、抛射、单体砸击，适合掷弹兵的单体伤害语义。
- 主体：短圆柱棋子。
- 顶部：TMP Text 写 `砲`，贴在上表面。
- 可加一个小圆球/石弹标记，暗示投掷。
- 颜色：橙黄、土红、石质感，和炮兵区分。
- 体型：比炮兵小一点。

语义：范围威胁。

### Crossbow / 弩手

目标：直线远程攻击。

已定方向：

- 国际象棋战车 / Rook 变体。
- Rook 本身有直线控制语义，和弩手的直线威胁匹配。
- 不是传统稳重棋子底座，而是尖底/楔形底座，暗示箭头和方向性。
- 上方保留城垛/塔楼轮廓。
- 可加细长箭杆或前向尖刺，强化弩塔/箭塔感。
- 颜色：冷蓝/青色，和玩家棋子、炮系敌人区分。
- 体型：高瘦、直线感强。

语义：线性威胁。

### Artillery / 炮兵

目标：更重型远程单位。

已定方向：

- 中国象棋炮系棋子，使用火字旁的 `炮`。
- `炮` 更现代、更火力化，适合重火力/高压单位。
- 主体：更厚、更大的短圆柱棋子。
- 顶部：TMP Text 写 `炮`，贴在上表面。
- 可加粗外圈或短炮管标记，强调重火力。
- 颜色：深红、紫红、火光边。
- 体型：比掷弹兵更大、更厚。

语义：慢、重、危险。

### CurseCaster / 咒术师

目标：控制/锁定核心箱子的特殊敌人。

已定方向：

- 麻将红中方向。
- 不使用“咒”字。`红中` 是真实牌面符号，对不认识汉字的人也更像符印/封印。
- 主体：竖立麻将牌或短厚牌块。
- 正面：红中符号。
- 可做成两个 `中` 竖向串联：从字形上接近 `串`，同时像锁链/封印 icon，适合“锁定核心箱子”的语义。
- 颜色：白/黑底 + 红色符号 + 紫色边光。
- 可加环形光圈、紫色锁链/连接线、悬浮感，强化控制/锁定语义。

语义：控制、诅咒、锁定。

## 敌人视觉体系总结

当前敌人不走写实兵人路线，而是“桌游/棋类/牌类规则件”路线：

- Go：围棋子
- Guokui：一筒麻将牌
- Ertong：二筒麻将牌
- Grenadier：`砲`，中国象棋炮系，石字旁
- Artillery：`炮`，中国象棋炮系，火字旁
- Crossbow：尖底国际象棋战车 / Rook
- CurseCaster：红中麻将牌

这套主题与当前“霓虹战术沙盘 / 电子棋盘”的画面语言一致。识别优先级是：轮廓 > 颜色 > 符号/文字。

### Wall / Unstable Wall

目标：障碍必须清楚，不能抢单位。

方向：

- 当前橙色电子墙可保留
- 不稳定墙加裂纹/闪烁/红色边线
- 普通墙和可破墙必须有明显材质差异

### Target / Core Target / Enemy Target

目标：地图语义点。

方向：

- 普通目标：低矮圆形/方形地面标记
- 核心目标：高一点的青白信标
- 敌人出生点：红/橙危险门或传送格

## 明日执行顺序

1. 先替换玩家、普通箱子、核心箱子
2. 再替换 Go、Grenadier、Crossbow 三个最常见敌人
3. 再替换 Guokui、Ertong、Artillery、CurseCaster
4. 最后处理 Wall / Target 的细节差异

## 验收标准

- 俯视 1 秒内能分辨玩家、箱子、敌人、目标
- Go / Grenadier / Crossbow 至少能通过轮廓分辨，不只靠颜色
- 核心箱子必须比普通箱子更重要
- 敌人出生点不要被误认为普通目标
- 模型不遮挡格子判断和路径预览
- Build 后模型和材质正常显示

## 暂不做

- 复杂骨骼动画
- 写实人物
- 高面数模型
- 大量粒子特效
- 会遮挡棋盘格的夸张模型
